# 🅿️ LOCIFY: Ecosistema Web para Gestión de Parqueaderos

## 1. Descripción
*Locify* es un ecosistema web integral diseñado para la gestión, control y verificación de parqueaderos. El proyecto busca optimizar la trazabilidad de vehículos y usuarios mediante una arquitectura moderna y escalable.

## 2. Tecnologías Implementadas
* **Frontend:** Next.js
* **Backend:** Node.js / API Routes
* **Autenticación:** Google OAuth 2.0
* **Control de Versiones:** Git y GitHub
